# Agentic-OS_Backdoor
Proyecto referente a la materia de Sistemas Operativos, a ser usado en Linux
#
El plan es embeber un “backdoor” en el API del sistema de ventanas (X11) de un sistema operativo POSIX compatible. Así cada vez que se cree una ventana y se escriba en ella, el texto sea enviado a un sistema de "IA" (Algoritmo a entrenar usa la tecnica llamada “bag of words”) para "entrenar un agente de IA en base al contexto del usuario".
